| **Module** | **Tên Module & Người phụ trách** | **Mô tả Chức năng & Công việc** |
| --- | --- | --- |
| **M01 & M08** | **Đỗ Kiến Hưng (Project Manager)** | **Xác thực & Bảo mật:** Đăng ký, Đăng nhập, Google OAuth2, JWT. **Đánh giá & Tương tác:** Viết đánh giá, chấm sao, hiển thị đánh giá kèm bộ lọc. |
| **M02** | **Quản lý Sản phẩm (Admin)** *(Người phụ trách: Võ Đức Hoàng)* | **Chức năng cũ:** CRUD sản phẩm, quản lý hình ảnh, thông số kỹ thuật, tồn kho. **Chức năng mới:** Bổ sung cơ chế **Soft Delete** (xóa mềm) để tránh mất dữ liệu. **Công việc:** Xây dựng API để quản lý sản phẩm. Thiết kế giao diện cho admin để thêm, sửa, xóa sản phẩm một cách trực quan, bao gồm cả trình upload ảnh. |
| **M03** | **Quản lý Danh mục & Thương hiệu (Admin)** *(Người phụ trách: **Huỳnh Ngọc Thạch**)* | **Chức năng cũ:** CRUD Danh mục và Thương hiệu. **Chức năng mới:** Tăng cường logic kiểm tra ràng buộc (ví dụ: không cho xóa thương hiệu nếu vẫn còn sản phẩm). **Công việc:** Phát triển các API và giao diện quản lý cho danh mục và thương hiệu. Đảm bảo giao diện dễ sử dụng và logic backend chặt chẽ. |
| **M04** | **Trải nghiệm Khám phá Sản phẩm (Client)** *(Người phụ trách: **Trần Quốc Giăng**)* | **Chức năng cũ:** Hiển thị danh sách sản phẩm, trang chi tiết, tìm kiếm. **Chức năng mới:**  1. Xây dựng chức năng **So sánh sản phẩm**. 2. Xây dựng hệ thống **Gợi ý sản phẩm** (Sản phẩm tương tự, Phụ kiện đi kèm). 3. Tích hợp **AI Chatbot** (phối hợp với vai trò AI Lead). **Công việc:** Chịu trách nhiệm các trang chính của người dùng. Phát triển các component React phức tạp cho việc lọc, sắp xếp, so sánh và gợi ý. |
| **M05** | **Giỏ hàng (Shopping Cart)** *(Người phụ trách: **Lưu Trần Kim Phú**)* | **Chức năng cũ:** Thêm/sửa/xóa sản phẩm trong giỏ hàng. **Chức năng mới:** Tối ưu hóa việc lưu trữ giỏ hàng bằng **Redis**, hỗ trợ đồng bộ giỏ hàng giữa các thiết bị khi đăng nhập và lưu giỏ hàng cho khách (guest). **Công việc:** Xây dựng API để tương tác với Redis. Phát triển component giỏ hàng trên frontend, đảm bảo cập nhật real-time và trải nghiệm mượt mà. |
| **M06 & M07** | **Huỳnh Hữu Huy** | **Thanh toán:** Tích hợp VNPay/Momo, quy trình checkout. **Quản lý Đơn hàng:** Tra cứu đơn hàng công khai, cập nhật trạng thái đơn hàng (Client & Admin). |
| **M09** | **Khuyến mãi & Voucher** *(Người phụ trách: **Nguyễn Văn Quang Duy**)* | **Chức năng cũ:** CRUD Voucher. **Chức năng mới:** Mở rộng logic voucher (giảm theo %, số tiền cố định, freeship). Cải thiện giao diện quản lý voucher cho admin. **Công việc:** Thiết kế lại cấu trúc bảng Voucher để linh hoạt hơn. Xây dựng API và giao diện cho admin để tạo và quản lý các chiến dịch khuyến mãi. |
| **M10** | **Quản lý Người dùng & Dashboard (Admin)** *(Người phụ trách: **Trần Thị Thanh Trang**)* | **Chức năng cũ:** Xem danh sách người dùng, khóa/mở khóa tài khoản. **Chức năng mới:** Xây dựng trang **Dashboard** cho Admin với các biểu đồ thống kê cơ bản (doanh thu, đơn hàng mới...) sử dụng thư viện như Chart.js. **Công việc:** Xây dựng các API tổng hợp dữ liệu thống kê. Thiết kế và phát triển trang Dashboard, trang quản lý người dùng cho admin. |
